## **常见问题**
### 1、运行时异常
```
{  
    "code": 4002,  
    "message": "获取标签服务失败，请检查是否有注入：TagService = null",  
    "errors": []  
}
```
**解决方式：**  
添加依赖：
>     <dependency>
>         <groupId>com.jfeat</groupId>
>         <artifactId>tag-service</artifactId>
>         <version>1.0.0</version>
>     </dependency>
